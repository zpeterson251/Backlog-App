# Backlog Project

## Overview
The Backlog project is a web application built with React that allows users to manage their saved games. Users can add, edit, delete, and sort their games based on various criteria. The application also supports searching through the game collection.

## Features
- Add new game entries
- Edit existing game details
- Delete game entries
- Filter games by status (e.g., Playing, Finished, Dropped)
- Sort games by title, release date, rating, and priority
- Group games by platform, region, franchise, or series
- Search functionality to find games quickly

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/backlog.git
   ```
2. Navigate to the project directory:
   ```
   cd backlog
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application
To start the development server, run:
```
npm start
```
The application will be available at `http://localhost:3000`.

### Building for Production
To create a production build, run:
```
npm run build
```
This will generate a `build` directory with the optimized application.
